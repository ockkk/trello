# trello_client
todo application인 trello의 기본 기능들을 클론한 프로젝트입니다. 
회원가입 탈퇴 수정 과 보드,컨테이너,카드의 추가,수정,삭제가 가능하며 카드의 컨테이너간 이동이 가능합니다.
